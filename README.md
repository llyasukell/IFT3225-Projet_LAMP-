# IFT3225 - Projet LAMP (Partage de Voyages)

Ce projet permet de créer et d'afficher des tuiles de voyage.

## Comment ça marche ?
- **PHP / MySQL** pour gérer les utilisateurs et les voyages.
- **JavaScript (Fetch)** pour afficher les tuiles sans recharger la page.
- **CSS** pour le design des tuiles et de la navigation.

## Installation rapide
1. Importe le fichier `database.sql` dans ton phpMyAdmin.
2. Les accès à la base de données sont dans `config.php`.

## Note sur le fichier config.php
Normalement, le fichier `config.php` ne devrait pas être partagé de cette façon, car ce n'est pas sécuritaire. Mais il est là car c'est juste un petit projet pour un devoir, afin de faciliter le test et la correction.
